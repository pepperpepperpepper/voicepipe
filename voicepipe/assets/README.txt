Assets Directory
================

recording_icon.tiff - Mail.App icon used for systray when recording is active.
                     This is a NeXTSTEP-style icon that fits well with the
                     voicepipe recording functionality.

test.mp3 - Short speech sample used by `voicepipe smoke`.
           Source: https://github.com/openai/whisper (tests/jfk.flac), converted
           to MP3 (mono, 16kHz, 64kbps).
test.expected.txt - Expected substring for the smoke transcript check.
